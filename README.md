# vueclass

## CLI 설치하기

npm install -g @vue/cli

## VUE 설치하기

vue create vueclass

## VUE 서버시작

vue run serve

## 파이어베이스 시작하기 - 설치 -> 프롬포트 열어서 제일 상위 폴더에 설치하기

npm install -g firebase-tools

## 로그인하기

firebase login
firebase init

## 빌드하는 법

npm run build
firebase deploy
